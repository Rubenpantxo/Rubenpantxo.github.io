Importar escaneo — preparado para el futuro

Una web estática no puede escanear tu red, pero la app ya define el formato. Cuando quieras, un script en un PC de casa podrá generar este JSON y lo importarás desde «⛁ Datos → Importar JSON»:

{
  "escaneo": [
    { "ip": "192.168.1.34",
      "mac": "A8:5E:45:1B:22:C3",
      "nombre": "PC principal",
      "red": "MOVISTAR-6688" }
  ]
}

Los aparatos existentes se casarán por MAC o IP y se actualizarán; los nuevos se añadirán a su red.
